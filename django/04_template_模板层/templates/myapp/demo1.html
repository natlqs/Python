1. 模板中的变量输出

  • {{name}}   {{age}}
  • {{stu.name}} {{stu.age}}
  • {{a.0}} {{a.1}} {{a.2}}

2. 模板中的标签使用

{% for da in dlist %} {% endfor %}
姓名 性别 年龄 状态
{{da.name}} {% if da.sex == 1 %}男{% else %}女{% endif %} {{da.age}} {% if da.state == 0 %} 管理员 {% elif da.state == 1 %} VIP账户 {% else %} 普通用户 {% endif %}
令牌：{% csrf_token %}

3. 模板中的过滤器使用

  • 原值：{{name}}
  • 大写：{{name|upper}}
  • 小写：{{name|lower}}
  • {{time|date:'Y-m-d H:i:s w'}}
  • {{a|join:":"}}

4. 模板中的运算

  • 两个数： {{m1}} {{m2}}
  • 求和： {{m1|add:m2}}
  • 求差： {{m1|add:-20}}
  • 相乘： {% widthratio m1 1 m2 %}
  • 相除： {% widthratio m1 m2 1 %}
